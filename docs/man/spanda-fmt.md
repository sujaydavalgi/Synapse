# spanda-fmt(1)

## NAME

fmt — Format Spanda source to canonical style.

## SYNOPSIS

```
spanda fmt [--json] <file.sd>
```

## DESCRIPTION

Format Spanda source to canonical style.

## OPTIONS

`--json` — report whether the file changed

## EXAMPLES

```bash
spanda fmt examples/rover.sd
```
