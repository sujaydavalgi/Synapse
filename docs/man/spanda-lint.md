# spanda-lint(1)

## NAME

lint — Run linter rules beyond parse/type checking.

## SYNOPSIS

```
spanda lint [--json] <file.sd>
```

## DESCRIPTION

Run linter rules beyond parse/type checking.

## OPTIONS

`--json` — structured lint report

## EXAMPLES

```bash
spanda lint robot.sd
```
